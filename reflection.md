# Reflection on Intelligent Agents Module

---

## WHAT

This module focused on the design and development of intelligent agent systems and their application in solving real-world problems. The main project involved designing an Intelligent Academic Agent System as part of a team, followed by an individual implementation of the system using Python.

The team project focused on defining the problem, identifying system requirements, selecting an appropriate agent architecture, and producing system diagrams. The Belief–Desire–Intention (BDI) model was chosen as the core architecture, as it supports goal-oriented and structured behaviour (Bratman, 1987).

My individual work involved implementing the system, including designing the modules, developing the data processing pipeline, and testing the system. The system was designed to retrieve, process, rank, and store academic data using a structured workflow.

Throughout the module, I also participated in discussion forums, where I explored different agent architectures and their applications. These discussions supported my understanding of intelligent systems and influenced the design decisions in the project.

---

## SO WHAT

This experience significantly improved my understanding of intelligent agent systems, particularly the difference between theoretical design and practical implementation. While the team project provided a strong conceptual understanding of system architecture and agent models, the individual implementation revealed the challenges of translating these ideas into a working system.

One of the key learning points was understanding how intelligent behaviour can be simulated within a system. The ranking mechanism implemented in the project demonstrated how an agent can evaluate and prioritise information rather than simply retrieving data. Although the logic was simple, it introduced the concept of decision-making within the system.

Another important learning outcome was the importance of modular system design. By separating the system into different components, such as search, extraction, processing, and storage, the system became easier to develop, test, and maintain. This reflects established principles in software engineering and intelligent systems design (Russell and Norvig, 2021).

Working in a team also contributed to my learning. The distribution of tasks and collaboration during the design phase helped me understand how complex systems are developed in a structured way. Communication through meetings and online tools ensured that all team members were aligned.

However, there were also challenges during the project. External factors, including instability in the region, affected consistency and focus. Despite this, the team was able to adapt and continue working effectively. This highlighted the importance of resilience and adaptability in professional environments.

From a critical perspective, the system has limitations. The use of simulated data instead of real-world data reduces its practical applicability. Additionally, the ranking algorithm is relatively simple and does not reflect advanced intelligent techniques such as machine learning. These limitations show that while the system demonstrates key concepts, further development is required for real-world deployment.

---

## NOW WHAT

This module has provided a strong foundation in intelligent agent systems and has highlighted areas for future development. One of the key areas I plan to improve is integrating real-world data sources into systems. This will involve learning how to use APIs and web scraping techniques to retrieve live data (Mitchell, 2018).

Another important area for development is improving the intelligence of the system. In future projects, I aim to explore machine learning techniques to enhance decision-making and ranking capabilities. This will allow the system to learn from data and improve its performance over time.

I also plan to continue developing my system design skills by working on more complex projects. This includes designing systems that can adapt to dynamic environments and handle larger datasets.

From a professional perspective, the module has improved my ability to work both independently and as part of a team. I will apply these skills in future academic and professional settings, particularly in areas related to artificial intelligence and cybersecurity.

Overall, this module has not only improved my technical knowledge but also enhanced my critical thinking, problem-solving, and adaptability. These skills are essential for developing intelligent systems and addressing real-world challenges.
