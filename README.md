# Intelligent Academic Agent System – E-Portfolio
# Ajayeb Mohammed S H Al-Bedewi - aalbedewi98

## 1. Introduction
This e-portfolio presents my work and learning development throughout the Intelligent Agents module. It documents the design, implementation, and evaluation of an Intelligent Academic Agent System developed in Python. The purpose of this portfolio is to demonstrate both technical and analytical skills, supported by evidence such as code, system outputs, and design artefacts.

The project focuses on solving the inefficiency of manual academic research by implementing an intelligent agent that automates the process of searching, extracting, processing, ranking, and storing research data. This portfolio also reflects my individual contributions within a team environment and highlights the professional and technical skills developed during the module.
